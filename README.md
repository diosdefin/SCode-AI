# SCode AI

## Описание проекта

SCode AI - это приложение для конвертации кода между различными языками программирования. Оно предоставляет графический интерфейс для выбора исходного и целевого языков, ввода кода и получения результата конвертации. Приложение поддерживает историю конвертаций и использует Python для выполнения некоторых операций конвертации.

## Функциональность

- Конвертация кода между языками: C++, C#, Go, Java, JavaScript, Python, Rust, TypeScript
- Графический интерфейс на базе Windows Forms
- Сохранение истории конвертаций
- Детекция языка кода
- Управление иконками для языков

## Требования

- **.NET 8.0** или выше (SDK для сборки и запуска)
- **Python 3.x** (для сервиса PythonService, если используется)
- **Windows** (поскольку приложение использует Windows Forms)

## Установка зависимостей

1. Установите .NET 8.0 SDK с [официального сайта Microsoft](https://dotnet.microsoft.com/download/dotnet/8.0).
2. Если используется PythonService, установите Python 3.x и необходимые пакеты (если применимо).

## Как запустить проект

1. Клонируйте репозиторий:
   ```
   git clone <URL репозитория>
   cd <папка проекта>
   ```

2. Откройте проект в Visual Studio или другом IDE, поддерживающем .NET.

3. Соберите проект:
   ```
   dotnet build
   ```

4. Запустите приложение:
   ```
   dotnet run --project Konvertor/Konvertor.csproj
   ```

   Или откройте Konvertor.sln в Visual Studio и нажмите F5 для запуска в режиме отладки.

## Структура проекта

- **Konvertor/**: Основной проект
  - **Controls/**: Пользовательские элементы управления (например, IconComboBox)
  - **Forms/**: Формы приложения (MainForm, HistoryForm, DarkMessageBox)
  - **Models/**: Модели данных (ConversionHistory, ConversionResult, Language)
  - **Services/**: Сервисы (CodeDetector, HistoryService, IconManager, PythonService)
  - **Resources/**: Ресурсы (иконки языков)
  - **Properties/**: Свойства проекта
- **Konvertor.sln**: Файл решения Visual Studio

## Сборка и публикация

Для публикации приложения:
```
dotnet publish Konvertor/Konvertor.csproj -c Release -r win-x64 --self-contained
```

Это создаст самодостаточный исполняемый файл в папке publish.
